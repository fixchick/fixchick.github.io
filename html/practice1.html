Анина практика по тегам!

сегодня Аня узнала много интересного о штмл, а именно о тегах, которые существуют в этом языке разметки. Ане показалось это очень интересным, поскольку с помощью таких тегов Аня может сделать себе простейшую страничку в интернете, снять это на теле и выложить в инстаграм, чтобы похвастаться своим подписчикам.

Итак, о каких же тегах узанала сегодня Аня. Давайте перечислим их.

  • h1 это заголовок, он может быть только один единственный
  • h2 следующий заголовок
  • h3 тоже заголовок
  • h4 еще один заголовок
  • h5 и снова заголовок
  • h6 последний заголовок
  • p чтобы писать с нового абзаца
  • span чтобы выделять текст
  • a targer href чтобы вставлять ссылки
  • src alt чтобы вставлять картинки
  • ul чтобы делать список через точку
  • ol чтобы делать нумерованый список
  • li чтобы делать отдельные строчки для списка
